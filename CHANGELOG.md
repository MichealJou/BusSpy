# Changelog

## v0.1.2

### 更新内容

- Windows release 构建隐藏控制台窗口，双击启动时不再同时弹出 cmd。
- 更新检测改为页面右上角提示条，不再用弹窗打断用户。
- 更新提示支持 `稍后提醒`、`忽略此版本`、`打开下载页`。
- `关于` 改为菜单入口，可分别选择关于 BusSpy、检查更新、版本协议、更新策略和打开发布页。
- `关于 BusSpy` 增加作者 Felix Chou、当前版本、平台支持和软件说明。
- 版本显示统一为 `vMAJOR.MINOR.PATCH`，并在状态栏和更新提示中展示。
- 外部链接改为调用系统默认浏览器打开，修复打开发布页/下载页无反应的问题。
- README 补充更新交互、版本协议和 macOS 打开未签名应用说明。

### 版本协议

BusSpy 使用 SemVer 语义化版本：

```text
MAJOR.MINOR.PATCH
```

- `MAJOR`：不兼容变更
- `MINOR`：兼容旧版本的新功能
- `PATCH`：问题修复和小优化
